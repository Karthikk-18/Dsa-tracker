# ⚡ DSA Tracker

A clean, no-backend web app to track your progress across **300 LeetCode problems** organized by topic.

## 🔗 Live Demo
> Replace with your GitHub Pages URL: `https://yourusername.github.io/dsa-tracker`

## 📦 Features

- 10 topics × 30 problems each (Arrays, Trees, Graphs, DP, and more)
- Click any problem → opens directly on LeetCode
- Check off solved problems with a ✅ tick
- Progress bars per topic + overall progress tracker
- Saves your progress in the browser (localStorage) — persists on refresh
- Enter your name on first visit

## 🚀 Usage

No install needed. Just open `index.html` in any browser — or visit the live link above.

> **Note:** Progress is saved per browser. Each person tracks their own progress independently.

## 🛠️ Built With

- Pure HTML, CSS, JavaScript
- No frameworks, no backend, no dependencies
